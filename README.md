# 💰 SmartMoney AI

An AI-powered personal finance assistant that helps users track their financial health, manage expenses, and get smart financial advice.

---

## 🚀 Problem Statement
Most people do not have proper financial planning and rely on guesswork for saving, budgeting, and investing. Financial advisors are expensive and not accessible to everyone.

---

## 💡 Solution
SmartMoney AI provides:
- 📊 Money Health Score based on income & expenses
- 🤖 AI Chatbot for financial guidance
- 📈 Simple and interactive UI for better understanding

---

## ✨ Features
- Calculate financial score instantly
- Get smart suggestions to improve savings
- Ask financial questions (budget, saving, investing)
- Clean and user-friendly interface

---

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- API: Custom AI logic (rule-based chatbot)
